Installation
============

Dependencies
^^^^^^^^^^^^
    - You need ``ffmpeg`` `installed <https://www.ffmpeg.org/download.html>`_ and in your PATH.
    - Only tested on Python 3.7.0


Install as global
-----------------
Install using pip::

    $ pip install pyvid

check installation::

    $ pyvid --version
    pyvid, version 0.0.7
